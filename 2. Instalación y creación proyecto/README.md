## Ejercicio 2:

Crea una clase llamada Curso que debe tener:
* Nombre (máximo 50 carácteres).
* Descripción Corta (máximo 280 carácteres).
* Descripción larga.
* Público Objetivo.
* Objetivos.
* Requisitos.
* Nivel (sólo puede ser Básico, Intermedio o Avanzado. para ello usa un enumerado).
